# 学大教育 | AI 全能情报指挥中心 V24 (原生搜索版)

## 项目简介
这是一个基于Web的AI内容生成工具，专为教育行业设计，用于快速生成短视频脚本和小红书笔记。

## 主要功能

### 🎯 核心功能
- **情报筛选**：支持按考试信息、学校资讯、家长咨询、艺考资讯、新媒体榜等分类浏览资讯
- **风格设定**：提供"深度观察员"和"金牌规划师"两种AI人设风格
- **一键生成**：快速生成短视频脚本和小红书笔记
- **封面设计**：自定义封面背景风格和标题
- **原生搜索**：支持直接搜索原文标题

### 📱 技术特性
- 响应式设计，支持桌面和移动设备
- 实时AI打字效果
- 支持多种背景渐变风格
- 一键复制生成的文案
- 支持封面图下载

## 技术栈

- **前端框架**：原生HTML/CSS/JavaScript
- **CSS框架**：Tailwind CSS
- **图标库**：Font Awesome
- **外部库**：
  - html2canvas (用于生成封面图)
  - FileSaver.js (用于下载文件)
- **字体**：
  - Noto Sans SC (中文)
  - JetBrains Mono (代码)

## 快速开始

### 本地运行
1. 克隆或下载项目文件
2. 在浏览器中直接打开 `index.html` 文件
3. 或使用本地服务器运行：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server -p 8000
   
   # 然后访问 http://localhost:8000
   ```

### GitHub Pages部署
1. 将项目推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择主分支和根目录
4. 访问生成的GitHub Pages URL

## 使用说明

1. **选择资讯**：在左侧选择一个资讯分类，然后点击一条资讯
2. **设置风格**：选择"深度观察员"或"金牌规划师"风格
3. **生成文案**：点击"立即生成"按钮
4. **查看结果**：在右侧查看生成的短视频脚本和小红书笔记
5. **自定义封面**：切换到"封面设计"标签，自定义封面风格和标题
6. **下载内容**：复制文案或下载封面图
